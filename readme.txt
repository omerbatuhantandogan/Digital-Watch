To run the program ;

After run it, you will see the menu and options for your purpose;

You may set time instead of automatically generated localtime,

You may set up to 10 alarms and stop them while alarm(s) is warning, once you stopped the alarm , you will have space to set another alarm

You may set timer up to 10000 seconds and start it with input "1" and you may use pause-resume functions by using input "2" or reset it by input "3"

You may show current time either automatically generated one or manually entered one, last saved one will be shown

You may close the program by entering input " 55555" in menu

You may initialize time as localtime by entering input "6" in menu
